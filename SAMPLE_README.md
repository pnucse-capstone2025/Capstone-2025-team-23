### 5. 설치 및 사용법

본 프로젝트는 macOS Sequoia 15.6.1 버전에서 개발되었으며 code/Swifood/ 폴더에서 터미널을 연 후 다음의 과정을 수행하여 실행할 수 있다. 코드 작성은 해당 폴더의 recipes.swifd 파일에 작성하면 된다.
```
$ make
$ ./swifood <recipe_name>
```
